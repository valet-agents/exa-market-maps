# Exa Market Maps

Describe the space — it builds the company list, founders, funding, and recent news, and keeps it fresh as the market moves.

## Prerequisites
- An [Exa](https://exa.ai) account with API access — you'll paste a key from `dashboard.exa.ai`
- A Slack workspace where you can install the agent's bot and invite it to one or more channels
